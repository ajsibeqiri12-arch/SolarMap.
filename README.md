# SolarMap Albania

SolarMap Albania is a digital platform that analyzes rooftops in Albania and estimates their solar energy potential.

## Features
- Roof analysis
- Solar energy estimation
- CO₂ reduction calculation
- Energy savings
- Interactive map

## Technologies
- Python / Flask
- OpenStreetMap API
- NASA POWER API
- HTML/CSS
- Leaflet.js

## Purpose
The project promotes renewable energy adoption in Albania and helps families understand the benefits of solar panels.
